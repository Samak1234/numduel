# NumDuel

A modular CLI-based number guessing game built in Python.

The project started as a simple monolithic script and is being progressively refactored into a scalable backend-oriented architecture to learn real software engineering concepts.

---

# Features

- Difficulty levels
- Hint system
- Score system
- Replay system
- Guess history tracking
- Modular architecture
- Input validation
- Git version control

---

# How To Run

```bash
cd numduel/game
python main.py
```
